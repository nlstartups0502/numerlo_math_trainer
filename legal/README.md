## For App Store Submission

When submitting to the App Store, you'll need to provide:

1. **Privacy Policy URL**: Use the GitHub Pages link once set up
   - Example: `https://yourusername.github.io/numerlo/privacy`

2. **Privacy Policy in App Store Connect**: 
   - Required field in App Store Connect
   - Must be accessible via a URL

## Hosting These Documents

### Option 1: GitHub Pages (Recommended)
1. Go to your repo Settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select your main branch
4. Your documents will be available at: `https://yourusername.github.io/your-repo-name/legal/privacy-policy` and `https://yourusername.github.io/your-repo-name/legal/terms-of-service`

### Option 2: Use the HTML versions
The `docs/` folder contains HTML versions that are better formatted for viewing in a browser.
